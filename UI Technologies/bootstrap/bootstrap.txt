# Boot Strap:
Bootstrap is the most commonly used framework for front end development.
Bootstrap provides several predefined libraries for CSS and java script.

getbootstrap.com


How to connect to bootstrap:
1. By using CDN (Content Delivery Network)
2. Locally

CDN:
